Complex Type mlhim243:DvCodedStringType
Namespace http://www.mlhim.org/xmlns/mlhim2/2_4_3
Annotations
 
A text item whose DvString-dv attribute must be the long name or description from a controlled
terminology. The key (i.e. the 'code') of which is the terminology-code attribute. In some cases, DvString-dv and
terminology-code may have the same content.
