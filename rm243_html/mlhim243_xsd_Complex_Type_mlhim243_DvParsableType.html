Complex Type mlhim243:DvParsableType
Namespace http://www.mlhim.org/xmlns/mlhim2/2_4_3
Annotations
 
Encapsulated data expressed as a parsable String. The internal model of the data item is not described
in the MLHIM model, but in this case, the form of the data is assumed to be  plaintext, rather than compressed or other types of large binary data. 
If the content is to be binary data then use a DvMedia.
